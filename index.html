    Bienvenue dans les pages "Guide Traduction" de Raisin Pi.

    L'objectif de ce tutoriel est de présenter une méthode de travail avec GitHub pour faciliter la traduction, collaboration et partage des ressources éducatives de la Fondation Raspberry Pi.

    Le contenu est formaté utilisant Markdown, select a template crafted by a designer, and publish. After your page is generated, you can check out the new gh-pages branch locally. If you’re using GitHub Desktop, simply sync your repository and you’ll see the new branch.

    Sections à Compléter

    Creer son compte sur Github Choisir un projet {repertoire} à traduire Choisir un fichier à travailler Editer le contenu Soumettre les modifications

    *Avancé *: Travailler "en local" Terminal Windows Mac

    "Tirer" le fichier à travailler Editer Markdown avec Atom (ou un autre outil de traitement) "Pousser" le fichier modifié
